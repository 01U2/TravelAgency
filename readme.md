# Travel Agency

## Goal:

Using Bootstrap & Grid System

## Project Overview:

- Create a responsive webpage for the landing page for a travel agency using Bootstrap components.

### Requirements

1. Header Section:
   - Include a navigation bar with the agency's name or logo aligned to the left.
   - Add navigation links (Home, Destinations, Tours, Contact) aligned to the right.

2. Hero Section:
   - Design a hero section featuring a background image of a scenic travel destination.
   - Place a heading, subheading, and a prominent call-to-action button at the center.

3. Destinations Section:
   - Create a grid layout displaying at least three travel destinations.
   - Use Bootstrap cards for each item, including an image, title, description, and a "Learn More" button.

4. Tours Section:
   - Implement Bootstrap’s accordion component to present different tour packages.
   - Each accordion item should include a title, brief description, and a "View Details" link.

5. Contact Form Section:
   - Design a contact form using Bootstrap form components.
   - Include fields for name, email, destination of interest, and a message.

6. Footer Section:
   - Create a footer containing copyright information and social media icons.

### Technical Requirements:

- Utilize Bootstrap components such as the navbar, jumbotron, cards, accordion, and form elements.
- Ensure the webpage is fully responsive.
- Test the layout on different screen sizes to ensure responsiveness.
- Additional Guidelines:

- Customize Bootstrap styles to match the travel agency theme.
- Use appropriate classes for responsiveness, such as container, row, col-md-6, etc.
- Enhance the design by incorporating relevant images and content.
